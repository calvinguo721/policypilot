# PolicyPilot - 政策通

> 全球AI政策匹配与申报辅助工具，开源建立信任

## 项目简介

政策通是一个开源的政策匹配与申报辅助工具，帮助企业和个人快速找到适用的AI政策补贴。

**核心理念**：
- 🔓 **开源透明** - 代码公开，任何人可审计
- 🤝 **信任驱动** - 用透明建立信任，用信任带来机会
- 🎯 **精准匹配** - AI驱动的政策匹配引擎

## 功能特性

- ✅ 政策数据采集与结构化
- ✅ 企业信息录入与条件匹配
- ✅ 资格诊断报告生成
- ✅ 申报材料模板与自动填充
- ✅ Web界面（[qiai.ren](http://qiai.ren)）
- ✅ API接口（供第三方接入）

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/calvinguo721/policypilot.git

# 安装依赖
pip install -r requirements.txt

# 运行
python main.py
```

📖 API 文档

完整的政策通 Token API 接入指南

Base URL: http://localhost:8002

🔐 认证方式

所有 API 请求需要在 Header 中携带 API Key 进行认证：

Headers
X-API-Key: your_api_key_here

🚀 快速开始

1

注册获取 API Key

调用注册接口，获取你的专属 API Key。

2

充值账户

通过充值接口为账户充值，基础查询 0.1 元/次，高级查询 0.5 元/次。

3

调用查询 API

使用你的 API Key 调用查询接口，开始享受政策数据服务。

💰 计费说明

查询类型 价格 限流 说明
基础查询 ¥0.1/次 100次/分钟 推荐
高级查询 ¥0.5/次 20次/分钟 含AI智能解读

⚠️ 预充值模式：账户余额 = 已充值 - 已消费。每次查询前检查余额是否充足。

🔌 API 端点

POST /api/token/register
注册新客户，获取 API Key
参数类型说明
company_namestring公司名称 必填
contactstring联系人
phonestring联系电话
emailstring邮箱
请求示例 (curl)
curl -X POST http://localhost:8002/api/token/register \
  -H "Content-Type: application/json" \
  -d '{"company_name": "测试公司", "phone": "13800138000"}'
请求示例 (Python)
import requests

resp = requests.post("http://localhost:8002/api/token/register", json={
    "company_name": "测试公司",
    "phone": "13800138000"
})
data = resp.json()
api_key = data["api_key"]
POST /api/token/recharge
为账户充值（模拟接口）
参数类型说明
amountnumber充值金额(元) 必填
请求示例 (curl)
curl -X POST http://localhost:8002/api/token/recharge \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -d '{"amount": 100}'
请求示例 (Python)
import requests

resp = requests.post("http://localhost:8002/api/token/recharge",
    headers={"X-API-Key": "your_api_key"},
    json={"amount": 100}
)
print(resp.json())
POST /api/token/query
基础政策查询 - 0.1元/次
参数类型说明
districtstring区域（如：海珠区、天河区）
industrystring行业
company_typestring企业类型
keywordsstring[]关键词数组
请求示例 (curl)
curl -X POST http://localhost:8002/api/token/query \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -d '{"district": "海珠区", "industry": "人工智能"}'
请求示例 (Python)
import requests

resp = requests.post("http://localhost:8002/api/token/query",
    headers={"X-API-Key": "your_api_key"},
    json={
        "district": "海珠区",
        "industry": "人工智能",
        "keywords": ["补贴", "创业"]
    }
)
policies = resp.json()["data"]["policies"]
POST /api/token/query/advanced
高级政策查询(AI智能解读) - 0.5元/次
参数类型说明
querystring自然语言查询 必填
company_infoobject企业信息（可选）
请求示例 (curl)
curl -X POST http://localhost:8002/api/token/query/advanced \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -d '{"query": "我们是海珠区的AI创业公司，有什么补贴可以申请？"}'
请求示例 (Python)
import requests

resp = requests.post("http://localhost:8002/api/token/query/advanced",
    headers={"X-API-Key": "your_api_key"},
    json={
        "query": "我们是海珠区的AI创业公司，有什么补贴可以申请？",
        "company_info": {"name": "测试公司", "employees": 20}
    }
)
result = resp.json()
print(result["data"]["response"])
GET /api/token/balance
查询账户余额
请求示例 (curl)
curl http://localhost:8002/api/token/balance \
  -H "X-API-Key: your_api_key"
响应示例
{
  "success": true,
  "balance": 99.50,
  "currency": "CNY"
}
GET /api/token/usage
查询用量统计
参数类型说明
daysnumber查询天数，默认30，最大365
请求示例 (curl)
curl "http://localhost:8002/api/token/usage?days=7" \
  -H "X-API-Key: your_api_key"
GET /api/token/bill
获取账单记录
响应示例
{
  "success": true,
  "bills": [
    {"date": "2024-01-15", "type": "recharge", "amount": 100},
    {"date": "2024-01-15", "type": "query", "amount": -0.1},
    {"date": "2024-01-15", "type": "advanced_query", "amount": -0.5}
  ]
}
GET /api/token/info
获取API基本信息
响应示例
{
  "success": true,
  "version": "1.0",
  "name": "政策通 Token API",
  "description": "政策数据Token化基础设施"
}

📡 API 端点

POST /api/token/register
注册新账户，获取 API Key
参数类型说明
company_namestring公司名称 必填
contactstring联系人
phonestring联系电话
emailstring邮箱
请求示例 (curl)
curl -X POST http://localhost:8002/api/token/register \
  -H "Content-Type: application/json" \
  -d '{"company_name": "测试公司", "phone": "13800138000"}'
Python SDK
from policypilot import PolicyPilotClient

client = PolicyPilotClient.register(
    company_name="测试公司",
    phone="13800138000"
)
print(client.api_key)
POST /api/token/recharge

  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -d '{"amount": 100}'

⚠️ 错误码说明

API 返回错误时，请根据错误码进行相应处理。以下是常见的 HTTP 状态码及业务错误码：

401 Unauthorized
API Key 无效或已过期
解决方案：请检查请求 Header 中的 X-API-Key 是否正确，或前往控制台重新获取 API Key。
402 Payment Required
余额不足，无法完成请求
解决方案：请前往控制台充值，基础查询需要 0.1 元余额，高级查询需要 0.5 元余额。
→ 立即充值
429 Too Many Requests
请求过于频繁，已触发限流
解决方案：请降低请求频率，基础查询限流 100次/分钟，高级查询限流 20次/分钟。
建议在客户端实现指数退避重试机制。
500 Internal Server Error
服务端内部错误
解决方案：可能是服务端临时故障，请稍后（建议 30 秒后）重试。
如果问题持续存在，请联系技术支持。
503 Service Unavailable
服务暂时不可用
解决方案：可能是例行维护或突发流量，请稍后重试。
Python SDK 错误处理示例
from policypilot import PolicyPilotClient, PolicyPilotError

client = PolicyPilotClient(api_key="your_api_key")

try:
    result = client.query_advanced(query="有什么补贴可以申请？")
except PolicyPilotError as e:
    if e.code == 401:
        print("API Key 无效，请检查")
    elif e.code == 402:
        print("余额不足，请充值")
        # 可以引导用户打开充值页面
        import webbrowser
        webbrowser.open("/console")
    elif e.code == 429:
        print("请求过于频繁，请稍后重试")
    else:
        print(f"其他错误: {e.message}")
网络超时重试示例
import time
from policypilot import PolicyPilotClient

client = PolicyPilotClient(api_key="your_api_key")

def query_with_retry(query, max_retries=3):
    for i in range(max_retries):
        try:
            return client.query_advanced(query=query)
        except Exception as e:
            if i == max_retries - 1:
                raise
            wait_time = 2 ** i  # 指数退避: 1s, 2s, 4s
            print(f"请求失败，{wait_time}秒后重试...")
            time.sleep(wait_time)
